# SubRosaUrlPatcher

Pathes Sub Rosa Executables to use custom master servers.

# Supported Versions:

- [ ] 38e
- [x] 34c
- [ ] 29a
- [ ] 25d

# Usage

```sh
node index.js path/to/exe hostname.net version
```
* Version is optional, defaults to 38e

thanks to @checkraisefold for showing me how this works
